ps_getavatar

说明

    根据phpsso uid获取头像url

resource ps_getavatar($uid)

参数

int $uid 用户id

返回值

array 四个尺寸用户头像数组

Example

<?php

$this->client->ps_getavatar($this->memberinfo['phpssouid']);

?>
